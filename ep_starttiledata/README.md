# ep_starttiledata

Reimplementation of Windows 10 Start menu components to keep it working on Windows 11 24H2 (starting from 26xxx.8493) and beyond. Based on StartTileData.dll of build 22621.1992 (part of Windows 11 22H2). Member, argument, and local variable names are based on SharedStartModel.dll of build 14361.

This project is a reverse-engineered source-level reconstruction derived from
analysis of publicly distributed Windows binaries and symbols. No leaked or confidential Microsoft source code was used.
